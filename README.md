# NearKart

## Project Overview

NearKart is a Hyperlocal Product Discovery Platform.

It helps users find products available in nearby local shops based on:
- Pincode
- City

Business owners can:
- Register using Mobile OTP
- Add Products
- Manage Offers
- Update Shop Details

Users can:
- Search Products
- Compare Prices
- View Nearby Shops
- Navigate to Shops
